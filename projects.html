R2D2

This is a repository for the Discord music bot. It is a python based discord music bot that operates 24/7 when turned on. This project utilised the discord apis libraries with the discord.ext module. This project was inspired by the gradual shut down of all the music bots on discord due to copy right issues, this led to a close down on popular discord bots.
